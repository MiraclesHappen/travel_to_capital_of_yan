

# 游览计划

1. 第一天
  1. 7:00于神州租车首体店集合
  1. 驱车前往易水湖畔农家院
  1. 安顿好车架，游湖，钓鱼，扯淡
  1. 晚上在农家院休息
1. 第二天
  1. 前往清西陵，游览至日头偏西
  2. 取道回京

# 费用

- 租车 （现代索纳塔8代 2.0 自动）
  - ¥402 x 2天 ≈ ¥800
- 油费
  - ¥300 
- 农家院
  - ¥100 x 2间 = ¥200
- 易水湖门票
  - ¥30 x 5人 = ¥150
- 易水湖船票
  - ¥240
- 清西陵通票
  - ¥120 x 5人 = ¥600
- 总计
  - (800 + 300 + 200 + 150 + 240 + 600) / 5 = 458
  - 加上吃喝每人消费应该在 500 左右
